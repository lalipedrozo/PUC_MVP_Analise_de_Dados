# PUC_MVP_Analise_de_Dados

Este repositório contém o MVP (Minimum Viable Product) desenvolvido para a pós-graduação em Análise e Ciência de Dados da **PUC-Rio**. O projeto consiste em uma análise exploratória completa, teste de hipóteses e tratamento de dados voltados para a geração de insights estratégicos.

## Definição do Problema
O objetivo deste trabalho é analisar o dataset [Nome do seu Dataset] para identificar padrões e correlações que impactam diretamente [O objetivo do seu projeto, ex: o comportamento de consumo / a precificação / a inadimplência]. 

A análise busca responder:
* Quais são as variáveis de maior impacto no resultado final?
* Existem padrões sazonais ou geográficos relevantes?
* Como a qualidade dos dados influencia a tomada de decisão?

## Estrutura do Projeto
O notebook está organizado seguindo o ciclo de vida de um projeto de dados:

1.  **Carga e Descrição dos Dados:** Importação e dicionário de variáveis.
2.  **Tratamento de Dados:** Limpeza, tratamento de valores ausentes e outliers.
3.  **Análise Exploratória (EDA):** Visualização de distribuições e matrizes de correlação.
4.  **Análise de Hipóteses:** Validação estatística e visual de suposições de negócio.
5.  **Conclusão e Insights:** Resumo dos achados e sugestões de próximos passos.

## Principais Insights
* **Hipótese 1:** [Resumo rápido, ex: Confirmada - a variável X possui correlação de 0.8 com Y].
* **Insight de Negócio:** [Ex: Identificou-se que o grupo A performa 20% melhor que o grupo B].
* **Qualidade dos Dados:** [Ex: O tratamento de outliers reduziu o ruído da análise em 15%].
